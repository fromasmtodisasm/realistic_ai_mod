Bink SDK 1.9c

HBINKBUFFER

基本データ型 HBINKBUFFER は、開いている BinkBuffer のハンドルです。

説明

HBINKBUFFER ハンドルは、BinkBufferOpen 関数から返されます。

また、BinkBuffer に関する主要なすべての情報を含む BINKBUFFER 構造体をポイントします。

BinkBufferOpen の呼び出しが失敗した場合、このハンドルは 0 になります。


グループ: 基本データ型
関連する関数: BinkBufferBlit, BinkBufferCheckWinPos, BinkBufferClose, BinkBufferGetDescription, BinkBufferLock, BinkBufferOpen, BinkBufferSetOffset, BinkBufferSetScale, BinkBufferUnlock
関連する構造体: BINKBUFFER
