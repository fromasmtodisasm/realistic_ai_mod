Bink SDK 1.9c

Open と Close

説明

Bink の Open 関数は、リソースを割り当てる関数です。Close 関数は、これらのリソースを解放する関数です。この関係を表す例には、BinkOpen と BinkClose があります。

Open 関数は、関連する他の API 関数で使用されるハンドルを返します。たとえば、BinkDoFrame を呼び出すには、BinkOpen から返されるハンドルが必要です。

Close 関数は、必ず Open 関数から戻されたいずれかのハンドルで呼び出します。これを守らないと、リソースがリークする場合があります。
