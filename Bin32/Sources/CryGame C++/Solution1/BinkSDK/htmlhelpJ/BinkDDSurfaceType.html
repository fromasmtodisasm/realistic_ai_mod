Bink SDK 1.9c

BinkDDSurfaceType

関数 BinkDDSurfaceType は、指定された DirectDraw サーフェスのフォーマットを表す値を返します。

BINK_COPY_FLAGS BinkDDSurfaceType(
  LPDIRECTDRAWSURFACE lpDirectDrawSurface
);

入力パラメータ

lpDirectDrawSurface
クエリするサーフェスを指定します。

戻り値

指定されたサーフェスのフォーマットを表す値を返します。

説明

この関数は、提供された DirectDraw サーフェスを表す値を返します。この関数を使用してサーフェス タイプを取得し、BinkCopyToBuffer 関数に渡すことができます。

サーフェスのタイプが未定義またはサポートされていない場合、この関数は -1 を返します。

DirectX 8 または 9 サーフェスの場合、それぞれ BinkDX8SurfaceType または BinkDX8SurfaceType 関数を使用し、BinkDDSurfaceType は使用しません。

MacOS では、BinkGDSurfaceType 関数を使用します。


グループ: BinkBuffer API
関連する関数: BinkCopyToBuffer, BinkCopyToBufferRect, BinkDX8SurfaceType, BinkDX9SurfaceType, BinkGDSurfaceType
関連する基本データ型: BINK_COPY_FLAGS, LPDIRECTDRAWSURFACE
