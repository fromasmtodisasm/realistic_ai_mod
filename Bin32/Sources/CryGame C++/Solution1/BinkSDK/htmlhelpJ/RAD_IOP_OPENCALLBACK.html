Bink SDK 1.9c

RAD_IOP_OPENCALLBACK

関数 RAD_IOP_OPENCALLBACK は、非同期のファイル オープンが完了したときに呼び出される関数をポイントします。

void RAD_IOP_OPENCALLBACK(
  RAD_IOP_ASYNC_OPEN_DATA const *open_data
);

入力パラメータ

open_data
RAD_IOPQueueAsyncOpenFile を呼び出したときの構造体へのポインタを指定します。

説明

この関数は、オープンの完了時に (インタラプト レベルで) 1 回呼び出されます。この関数の中に作成するコードは、必ずインタラプト セーフにします。

このファイル ハンドルを BinkOpen 関数で使用して IOP RAM から Bink ムービーを再生したり、AIL_open_stream 関数で使用して IOP RAM からサウンド ファイルを再生することができます。

このコールバックは成功と失敗のどちらの場合も実行されるため、status フィールドの値を確認して、オープンが実際に正常に行われたかどうかを判別してください。


グループ: ソニー PS2 用 RAD_IOP API
関連する基本データ型: void
関連する構造体: RAD_IOP_ASYNC_OPEN_DATA
