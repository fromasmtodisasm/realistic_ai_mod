Bink SDK 1.9c

BinkSoundUseSoundManager

BinkSoundUseSoundManager 関数は、サウンド出力システムとして MacOS Sound Manager を使用するように Bink に指示します。

S32 BinkSoundUseSoundManager();

戻り値

Bink の出力レイヤが Sound Manager に切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、MacOS Sound Manager をオーディオ出力として使用するように Bink に指示します。通常はこの関数を呼び出す必要はありません。Mac では BinkOpen の中で自動的に呼び出されます。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。


グループ: Bink のオーディオ システムのインターフェイス関数
関連する関数: BinkOpen, BinkSetSoundSystem, BinkSoundUseAX, BinkSoundUseDirectSound, BinkSoundUseLibAudio, BinkSoundUseMiles, BinkSoundUseMusyX, BinkSoundUseNDSSound, BinkSoundUsePSPSound, BinkSoundUseRAD_IOP, BinkSoundUseSDLMixer
関連する基本データ型: S32
