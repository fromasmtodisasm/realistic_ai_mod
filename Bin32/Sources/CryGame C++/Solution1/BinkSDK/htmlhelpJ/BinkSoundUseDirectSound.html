Bink SDK 1.9c

BinkSoundUseDirectSound

BinkSoundUseDirectSound 関数は、DirectSound をサウンド出力システムとして使用するように Bink に指示します。

S32 BinkSoundUseDirectSound(
  LPDIRECTSOUND lpDirectSound
);

入力パラメータ

lpDirectSound
DirectSound の関数 CreateDirectSound が返す DirectSound のオブジェクト ハンドルを指定します。

戻り値

Bink の出力レイヤが DirectSound に切り替わったかどうかを返します (正常に終了した場合は 0 以外、失敗した場合は 0)。

説明

この関数 (実際はマクロ) は、DirectSound をオーディオ出力として使用するように Bink に指示します。

この関数はグローバルに機能するので、呼び出す必要があるのは 1 回だけです。ただし、最初に BinkOpen を呼び出す前に呼び出さなければ、この関数は失敗します。


グループ: Bink のオーディオ システムのインターフェイス関数
関連項目: Win32 でのサウンド, Win64 でのサウンド
関連する関数: BinkOpen, BinkSetSoundSystem, BinkSoundUseAX, BinkSoundUseLibAudio, BinkSoundUseMiles, BinkSoundUseMusyX, BinkSoundUseNDSSound, BinkSoundUsePSPSound, BinkSoundUseRAD_IOP, BinkSoundUseSDLMixer, BinkSoundUseSoundManager, BinkSoundUseWaveOut
関連する基本データ型: LPDIRECTSOUND, S32
