Bink SDK 1.9c

BinkService

BinkService 関数は、Bink がサウンド システムにサービスを提供するための明示的なアイドル時間を提供します。

void BinkService(
  HBINK bink
);

入力パラメータ

bink
Bink ハンドルを指定します。

説明

この関数は、バックグラウンドの Bink サウンド サービスを明示的に実行します。CPU を極端に使用するタスクのためにサウンドがとぎれる場合は、この関数を一定間隔で呼び出すことを考慮してください。


グループ: Bink API
関連する基本データ型: HBINK, void
