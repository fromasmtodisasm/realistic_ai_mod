ソニー PS3 ビルド環境への Bink の組み込み

説明

Bink Video を使用するアプリケーションをビルドするには、ビルド プロセスに Bink SDK を含めます。それには、次の 3 つの簡単な手順に従います。

1) プロジェクトのインクルード ディレクトリとライブラリ ディレクトリに Bink SDK ディレクトリを追加します。

2) ソース ファイルで Bink.h をインクルードします。

3) スタティック ライブラリ BinkPS3.a にリンクします。Bink はアプリケーションに直接リンクされます。

以上です。これで Bink を使用できるようになりました。
