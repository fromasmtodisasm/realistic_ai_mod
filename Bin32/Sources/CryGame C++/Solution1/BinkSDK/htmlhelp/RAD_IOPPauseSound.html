RAD_IOPPauseSound

The function RAD_IOPPauseSound pauses a previously registered sound callback.

S32 RAD_IOPPauseSound(
  RAD_IOP_SOUND_DATA *sound_data
);

Out Parameters

sound_data
Specifies the sound callback to control.

Returns

Returns non-zero on success, or zero on failure.

Discussion

This function tells RAD_IOP to stop calling your sound callback.
