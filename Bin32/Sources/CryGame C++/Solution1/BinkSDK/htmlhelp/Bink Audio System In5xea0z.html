Bink SDK 1.9c

Bink Audio System Interface Functions

Discussion

To make Bink as easy to integrate as possible, we supply pre-written audio interfacing code on all of the platforms that we support. Some platforms even have multiple sound interfaces to support different APIs and hardware!

You can also write your own interfacing code, if none of the sound interface options works for your game (this is kind of complicated, so contact us for details), but this is rarely, rarely done.

Bink Audio System Interface Functions
BinkSoundUseMiles tells Bink to use the Miles Sound System as its sound output system.
BinkSoundUseLibAudio tells Bink to use LibAudio as its sound output system on the Sony PS3.
BinkSoundUsePSPSound tells Bink to use libwave or simple audio as its sound output system on the Sony PSP.
BinkSoundUseDirectSound tells Bink to use DirectSound as its sound output system.
BinkSoundUseWaveOut tells Bink to use the deprecated waveOut audio interface as its sound output system.
BinkSoundUseXAudio tells Bink to use XAudio as its sound output system on the Xbox 360.
BinkSoundUseRAD_IOP tells Bink to use the RAD_IOP API (which abstracts the IOP chip on the PS2).
BinkSoundUseSoundManager tells Bink to use the MacOS Sound Manager as its sound output system.
BinkSoundUseNDSSound tells Bink to use the Nintendo DS sound hardware as its sound output system.
BinkSoundUseAX tells Bink to use AX as its sound output system on the Nintendo GameCube or Wii.
BinkSoundUseMusyX tells Bink to use MusyX as its sound output system on the Nintendo GameCube.
RADARAMCALLBACKS specifies a set of replacement audio RAM memory management functions for use with AX on the Nintendo GameCube or Wii.
RADARAMALLOC points to a function that will allocate audio memory on Bink's behalf when using AX on the Nintendo GameCube or Wii.
RADARAMFREE points to a function that will free memory allocated by a previous call to the RADARAMALLOC function.
BinkSoundUseSDLMixer tells Bink to use SDL_mixer as its sound output system under GNU/Linux.
BinkSetSoundSystem tells Bink to use a custom sound provider as its sound output system.
