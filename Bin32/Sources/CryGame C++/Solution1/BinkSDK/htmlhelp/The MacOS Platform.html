The MacOS Platform

Discussion

The second operating system that Bink supported was MacOS. MacOS is strange to develop under - some pieces are advanced, some pieces seem crazily backwards. Bink is now developed under MacOS X and include Mach-O versions of the library for both Intel and PowerPC Macs. We also supply classic MacOS and Carbon versions of the Bink SDK.

Bink is extremely easy to use and integrate under this platform. This chapter will discuss everything you need to know!
