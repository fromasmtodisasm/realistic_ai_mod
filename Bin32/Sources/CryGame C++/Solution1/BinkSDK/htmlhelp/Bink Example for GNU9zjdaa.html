Bink Example for GNU/Linux

Discussion

The Bink SDK supplies a simple example for GNU/Linux that uses the SDL library.

To build this example, follow these steps:
1) Compile examsdl.c with GCC. You need to use the SDL include directories to build it (sdl-config can do this for you automatically).
2) Link the example with the examsdl object file, the Bink library (binklnx.a), the SDL and pthreads shared libraries, and the GCC glibc library (of course).

And that's it! You should be able to run the Bink example with a filename on the command line.
