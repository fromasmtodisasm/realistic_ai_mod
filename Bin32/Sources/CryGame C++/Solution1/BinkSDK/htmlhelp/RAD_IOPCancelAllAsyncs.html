RAD_IOPCancelAllAsyncs

The function RAD_IOPCancelAllAsyncs cancels all previous asynchronous file opens and reads.

S32 RAD_IOPCancelAllAsyncs();

Returns

Returns how many async operations were cancelled (if zero, then that usually means it had already completed).

Discussion

This function allows you to cancel all previously queued file open and read requests. Note that callbacks are still made when an open request is cancelled - make sure you check the status field.
