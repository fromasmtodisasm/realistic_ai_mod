HBINK

The basic type HBINK is a handle to an opened Bink file.

Discussion

A HBINK handle is returned from the BinkOpen function.

It also points to a BINK structure which contains all sorts of interesting information about the open Bink file.

If the BinkOpen call failed, then this handle will be zero.
