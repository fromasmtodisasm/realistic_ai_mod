Bink SDK 1.9c

BinkGetPalette

The function BinkGetPalette returns the global Bink palette for the BINKSURFACEP8 surface type.

void BinkGetPalette(
  VOID_PTR dest_addr
);

In Parameters

dest_addr
Address of 1024 bytes to receive the global Bink palette.

Discussion

This function returns the global palette that Bink uses for dithering BINKSURFACEP8 surface types. The palette is always returned in the simple format: B byte, G byte, R byte, pad byte, etc.
