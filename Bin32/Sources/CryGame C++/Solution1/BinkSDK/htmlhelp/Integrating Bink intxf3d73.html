Integrating Bink into a GNU/Linux Build Environment

Discussion

To build an application that uses Bink Video, you must include the Bink SDK in the build process. There are two simple steps to do this:

1) Include bink.h in your source files. You may want to add an directory include to your gcc compile line in your make file.

2) Link with the binklnx.a library file (just add this to your link line in your make file).

And that's it - you should now be able to use Bink!
