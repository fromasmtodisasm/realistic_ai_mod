Bink SDK 1.9c

BinkService

The function BinkService gives some explicit idle time to Bink to service its sound system.

void BinkService(
  HBINK bink
);

In Parameters

bink
Specifies the Bink handle.

Discussion

This function will perform background sound Bink services explicitly. This function can be called periodically when you have an extremely CPU-intensive task that is causing the sound to break up.
