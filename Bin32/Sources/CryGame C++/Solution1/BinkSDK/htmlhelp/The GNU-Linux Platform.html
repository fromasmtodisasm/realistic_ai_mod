The GNU/Linux Platform

Discussion

Bink fully supports GNU/Linux for x86. Linux is easy to develop under - good command line tools and a fine debugger. Bink was developed under RedHat 9, but should work on any modern distribution (glibc 2.2 and up). Bink is supplied as a static library on GNU/Linux.

Bink is extremely easy to use and integrate under this platform. This chapter will discuss everything you need to know!
