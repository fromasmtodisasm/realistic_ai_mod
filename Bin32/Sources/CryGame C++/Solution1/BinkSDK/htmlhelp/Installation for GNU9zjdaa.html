Bink SDK 1.9c

Installation for GNU/Linux

Discussion

The Bink SDK is provided as a compressed zip file that you download from our ftp site. You simply have to decompress it into a directory to use it.

Once installed, you can access this documentation online by clicking on the index.html file in the html folder. You may want to make a link to this file.
